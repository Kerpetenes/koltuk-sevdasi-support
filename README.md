## Sıkça Sorulan Sorular

### Reklamları nasıl kaldırırım?
Oyun içinde **Ayarlar** → **Reklamları Kaldır** butonuna basarak tek seferlik satın alma yapabilirsiniz.

### Satın alma yaptım ama reklamlar hala çıkıyor?
Ayarlar → **Satın Almaları Geri Yükle** butonuna basın.

### Oyun çöktü / hata aldım
Lütfen email ile bize ulaşın, cihaz modelinizi ve iOS sürümünüzü belirtin.
